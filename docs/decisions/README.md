# Architecture Decision Records

This folder contains lightweight ADRs documenting key decisions made during the project.

## Format

Each file follows the pattern `{number}-{title}.md` with sections:
- **Context** — Why was this decision needed?
- **Decisions** — What was decided and why?
- **Alternatives considered** — What was rejected?
